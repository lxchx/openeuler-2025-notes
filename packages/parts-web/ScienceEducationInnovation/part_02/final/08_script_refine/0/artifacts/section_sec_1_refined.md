# 研究方向总览：面向人工智能的存储 (Storage for AI)

本研究聚焦于机器学习系统与存储系统的交叉领域，核心方向为“面向人工智能的存储 (Storage for AI)”。随着人工智能模型与应用的发展，存储系统在其中扮演着愈发关键的角色。当前工作主要围绕以下三个层次展开：

### 1. 参数存储 (Parameter Storage)

随着人工智能模型（尤其是大模型）的规模持续增大，其需要存储的参数数量也随之急剧增多。因此，高效、可扩展的参数存储是支撑大规模模型训练与推理的基础。

### 2. 向量存储 (Vector Storage)

人工智能模型通过将世界万物表达为高维向量来进行理解和处理。无论是文本、图像还是其他模态的数据，最终都会转化为向量形式。因此，对这些向量进行高效存储、检索和管理是构建现代 AI 应用的关键组件。

### 3. 记忆存储 (Memory Storage)

记忆是区分智能体 (Agent) 与简单聊天机器人 (Chatbot) 的核心能力。为了让大型语言模型 (LLM) 能够联系上下文、持续学习并执行复杂任务，必须为其构建记忆机制。我们的研究将面向特定场景，探索面向 LLM 的记忆存储方案，具体技术路径包括：

- **检索增强生成 (RAG, Retrieval-Augmented Generation)**
- **参数化记忆方案**
