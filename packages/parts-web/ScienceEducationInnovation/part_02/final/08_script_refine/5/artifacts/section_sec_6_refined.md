# SSD层参数存储：MaxEmbed

## 1. 问题：传统预取方法在推荐模型中的局限性

传统的预取（Prefetching）方法在应用于推荐模型时，并不能达到预期效果。其主要瓶颈在于会导致 SSD 层出现严重的读放大（Read Amplification）问题，从而限制了有效带宽。

## 2. 解决方案：MaxEmbed 方法

针对上述问题，我们设计并提出了 MaxEmbed 方法。其核心机制与目标如下：

*   **核心机制**：选择性地为高频访问的“热点”参数创建副本。
*   **设计目的**：
    *   利用参数副本捕获数据中多种复杂的共现关系。
    *   从根本上解决 SSD 层的读放大问题。
    *   最终实现有效带宽的显著提升。
