# 专题分享：人工智能端到端的复杂攻击检测技术研究

## 1. 讲者简介
**李锭**
*   **机构**：北京大学
*   **研究领域**：人工智能安全、端到端攻击检测技术

## 2. 项目背景与社区合作
本项目致力于解决人工智能广泛应用背景下的安全防护挑战，并与 **openEuler 社区** 保持着长期的深度合作：
*   **合作起点**：2023 年，项目正式入驻社区并进行首次技术分享。
*   **当前进展**：2024 年，项目进入新阶段，本次分享将重点展示该研究课题的最新技术突破与进展。

## 3. 核心课题概述
**主题**：《人工智能端到端的复杂攻击检测技术研究》

**核心目标**：
在 AI 应用日益普及的趋势下，构建从端到端的全链路安全防御体系，针对复杂攻击行为实现高效、精准的自动化检测，确保底层系统与上层应用的安全性。
