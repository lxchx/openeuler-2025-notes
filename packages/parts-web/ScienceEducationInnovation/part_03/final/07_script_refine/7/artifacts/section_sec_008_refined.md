# SysArmor：开源系统安全原型及其演进路线

## 1. 系统概述与核心优势

**SysArmor** 是一个已在 **openEuler** 社区开源的系统安全原型，旨在提供高效、智能的端到端攻击检测能力。

### 1.1 性能表现
与行业主流工具（如 Sysdig、Linux Audit）相比，SysArmor 在资源开销上实现了显著优化，其**开销降低了一个数量级**，更适用于对性能敏感的生产环境。

### 1.2 核心功能
- **端到端检测**：覆盖从威胁感知到响应的完整攻击链路。
- **大模型集成**：支持通过大语言模型（LLM）进行交互式查询，用户可直接询问攻击详情，降低了安全分析的门槛。

除了 SysArmor，团队在系统安全领域还构建了多维度的研究工具：

| 工具名称 | 技术领域 | 功能描述 |
| :--- | :--- | :--- |

## 3. 未来演进与社区规划

### 3.1 社区迭代与数据共享
- **持续迭代**：依托 openEuler 社区，对 SysArmor 系统进行长期的功能优化与版本更新。
- **数据集公开**：计划通过隐私保护系统等合规方式公开研究数据集，旨在支持社区算法的共同迭代与验证。

### 3.2 跨领域应用扩展
团队正致力于将 SysArmor 的核心算法迁移并扩展至更多高价值领域：
- **大模型安全**：探索针对 LLM 基础设施的安全防护。
- **金融反欺诈**：利用底层系统行为分析提升金融业务的风险防控能力。
