# 总结与Q&A

## 技术核心总结

随着大模型并行方案（如专家并行）的应用，通信开销逐渐成为推理性能的主要瓶颈。为解决此问题，本次分享介绍了一系列基于基础通信库的优化方法，核心技术点如下：

- **异构多平面传输**：利用多种网络平面进行数据传输，提升整体带宽和效率。
- **M2N 通信模式**：将传统的点对点通信优化为多对多（M2N）通信模式，通过多通道叠加的方式提升性能。
- **计算与通信掩盖**：在执行计算任务的同时进行通信操作，以隐藏通信延迟，从而提升设备利用率和端到端性能。

该基础通信库能够高效实现 `reduce`、`all-gather`、`all-to-all` 等集合通信语义，为上层应用提供加速能力。
