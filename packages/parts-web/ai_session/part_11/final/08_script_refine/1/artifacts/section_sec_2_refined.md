# CTyunOS 智能安全中心概览

## 1. 平台简介

CTyunOS 智能安全中心是由天翼云操作系统团队开发的一款智能化漏洞全生命周期管理平台。该平台旨在实现对漏洞从感知、播报、分析、修复、测试到公告的全流程智能化跟踪与管理。

平台的核心特色是引入 AI 大模型能力，能够基于 CVE 信息、源码、补丁等多维要素，智能分析并精准评估漏洞对 CTyunOS 操作系统的实际影响。

## 2. 核心能力与技术基础

### AI 驱动的漏洞分析

- **智能评估**：集成 AI 大模型（如 DeepSeek），分析漏洞是否对操作系统构成影响。
- **数据驱动**：分析过程综合利用 CVE 信息、源码信息、补丁信息等多种数据源，以提高评估的准确性。

### 基于开源项目的扩展

CTyunOS 智能安全中心基于天翼云贡献至 openEuler 社区的开源项目进行扩展，以更好地服务于操作系统安全任务。主要包括：

- **`CVE-ease`**：用于漏洞处理与分析。
- **`security-baseline`**：用于安全基线管理。

## 3. 系统架构

系统采用前后端分离的现代化架构，并支持容器化一键启动部署。

- **前端技术栈**
  - **框架**: Vue.js
  - **图表**: ECharts
  - **托管**: Nginx

- **后端技术栈**
  - **框架**: Flask
  - **数据库**: PostgreSQL
  - **缓存**: Redis
  - **异步任务**: Celery
  - **智能分析**: 集成 DeepSeek 大模型

- **核心模块**
  - **CVE-Ease**：漏洞处理模块
  - **Security-Baseline**：安全基线管理模块

## 4. 功能总结

CTyunOS 智能安全中心是一个集成了 AI 大模型的智能化漏洞全生命周期管理平台。它通过自动化分析 CVE、源码及补丁信息，精准评估漏洞对 CTyunOS 的影响并追踪修复流程。同时，平台结合了播报管理、定时任务与权限控制等功能，实现了对漏洞从感知、分析到修复、公告的一体化智能管理。
