# 破解“医疗孤岛”：基于 openEuler 的新一代 CDSS 智能引擎实践

## 1. CDSS 概念介绍

CDSS（Clinical Decision Support System），即临床决策辅助智能系统，是一种面向医护人员的 AI 应用，旨在其临床工作中提供辅助决策。

在医疗这一严谨的业务场景下，CDSS 的主要应用价值包括：

- **规范化诊疗**
- **智能预警**
- **相关推荐**

## 2. 分享议程

本次分享将围绕以下四个方面展开：

1.  **医疗业务场景的困境**：分析当前医疗行业面临的挑战。
2.  **新一代 CDSS 智能引擎的构想**：针对上述困境，提出新一代引擎的设计理念。
3.  **基于 openEuler 的 Intelligence 知识库实践**：介绍为满足电子病历评级四级标准而构建的知识库。
4.  **自研 IntelliCore 决策引擎**：介绍为应对电子病历评级五级至八级等高级决策场景而自研的决策引擎。
